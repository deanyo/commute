# commute

Live train departures for my commute.

**Live:** [dnyo.co.uk/commute](https://dnyo.co.uk/commute/)

## Features

- Real-time National Rail departures
- Custom routes via URL: `/EUS/MKC`
- Platform predictions for Waterloo
- Click header to edit route inline
